# Node.js + React App

This project is a basic, responsive starter template for a Node.js app with a React frontend and Express backend.

## Structure
- `client/` — React app (frontend)
- `server/` — Express app (backend)

## Getting Started

### Client
1. Navigate to `client` and run `npm install`.
2. Start the React app with `npm start`.

### Server
1. Navigate to `server` and run `npm install`.
2. Start the server with `npm start`.

---

This template is ready for further customization.
